# 🛡️ Adventure Guild Mobile App

![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)
![Dart](https://img.shields.io/badge/dart-%230175C2.svg?style=for-the-badge&logo=dart&logoColor=white)
![UTS](https://img.shields.io/badge/Project-UTS_Mobile-orange?style=for-the-badge)

A premium, world-class mobile application built for a university assignment (UTS). This app features a dark RPG-themed aesthetic, robust input validation, and dynamic state management for an immersive "Adventure Guild" experience.

---

## ✨ Key Features

- **Character Registration**: A secure registration flow with strict validation for Name, Level (Numeric), Rank (Alphabetic), and Role Class.
- **Premium Dark UI**: Implemented with **Glassmorphism**, neon glows, and **Poppins** typography for a modern, high-end feel.
- **Dynamic Quest System**: 
  - Real-time state management to "Take" or "Complete" quests.
  - Smooth **Hero Animations** transitioning from the home list to quest details.
  - Interactive **SliverAppBar** with stretch effects.
- **Responsive Layout**: Designed mobile-first to ensure usability across different screen sizes.
- **Custom Assets**: High-quality fantasy artwork integrated directly into the application assets.
